# lgpt-resources
Resources for Little Piggy Tracker

# contents (version 1.2)
lgpt_* Tracks composed as part of OneOhFive compo, celebrating the 1.5.0 release of Little Piggy Tracker

        A selection of songs representing the versatility of the tracker

samplelib - a selection of samples to get you started composing

        IR-s: necessary if using PrintFX, a folder of [Impulse Responses](https://en.wikipedia.org/wiki/Convolution) to apply reverb on the sample
        Oscs: A selection of single-cycle waveforms. Set the loop mode to Osc to use
            Contents of metempsychosis produced by [Nine-H](https://github.com/Nine-H/metempsychosis)
